## <h1 className="bg-gradient-to-r from-blue-500 to-purple-500 inline-block text-transparent bg-clip-text text-5xl pt-2">React Install</h1>

<Steps>
  <Step>
    রিয়াক্ট জে এস ইন্সটল করা এবং আপনার কোড এডিটর সেটআপ করা।
    
    রিয়াক্ট ইনস্টল করার জন্য প্রথমে একটি কোড এডিটর ইনস্টল করুন। জনপ্রিয় কোড এডিটরগুলোর মধ্যে Visual Studio Code (VS Code) খুবই জনপ্রিয়। এটি ইন্সটল করার পর, আপনি রিয়াক্ট প্রজেক্ট তৈরি করতে প্রস্তুত।
  </Step>

  <Step>
    এরপর, কমান্ড প্রম্পট বা টার্মিনাল খুলুন এবং নিচের কমান্ডটি ব্যবহার করুন রিয়াক্ট ইন্সটল করতে:

    ```bash
    npm create vite@latest --template react
    ```

    এই কমান্ডটি Vite নামক একটি টুল ব্যবহার করে নতুন রিয়াক্ট প্রজেক্ট তৈরি করবে। Vite হল একটি দ্রুত ডেভেলপমেন্ট পরিবেশ যা রিয়াক্টের জন্য উপযুক্ত।
  </Step>

  <Step>
    কমান্ডটি রান করার পর, আপনাকে আপনার প্রজেক্টের নাম উল্লেখ করতে বলা হবে। আপনার পছন্দ অনুযায়ী একটি নাম দিন এবং এন্টার চাপুন। উদাহরণস্বরূপ, আপনি `my-react-app` নাম দিতে পারেন।

    এরপর, প্রজেক্ট তৈরি হয়ে গেলে, প্রজেক্টের ডিরেক্টরিতে প্রবেশ করুন:

    ```bash
    cd my-react-app
    ```

  </Step>

  <Step>
    অবশেষে, আপনার রিয়াক্ট অ্যাপ্লিকেশন চালু করতে নিচের কমান্ডটি ব্যবহার করুন:

    ```bash
    npm install
    npm run dev
    ```

    `npm install` কমান্ডটি প্রয়োজনীয় প্যাকেজগুলো ইনস্টল করবে এবং `npm run dev` কমান্ডটি আপনার অ্যাপ্লিকেশনকে লোকাল সার্ভারে চালু করবে। আপনার ব্রাউজারে গিয়ে `http://localhost:5173` এ ভিজিট করে আপনার নতুন রিয়াক্ট অ্যাপ্লিকেশন দেখতে পারবেন।
  </Step>
</Steps>

### Command Follow-Up

আপনার প্রজেক্ট সফলভাবে তৈরি হওয়ার পর, সবগুলো স্টেপ অনুসরণ করুন এবং উপরের কমান্ডগুলো চালিয়ে আপনার রিয়াক্ট অ্যাপ্লিকেশন রান করুন। যদি সবকিছু ঠিকঠাকভাবে হয়, তাহলে আপনি আপনার ব্রাউজারে রিয়াক্ট অ্যাপ্লিকেশন দেখতে পাবেন।
