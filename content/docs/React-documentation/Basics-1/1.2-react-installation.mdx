## <h1 className="bg-gradient-to-r from-blue-500 to-purple-500 inline-block text-transparent bg-clip-text text-5xl pt-2">React Install</h1>

---

<Steps>
  <Step>
    রিয়াক্ট জে এস ইন্সটল করা এবং তোমার কোড এডিটর সেটআপ করা।
    
    রিয়াক্ট ইনস্টল করার জন্য প্রথমে একটি কোড এডিটর ইনস্টল করো। জনপ্রিয় কোড এডিটরগুলোর মধ্যে Visual Studio Code (VS Code) খুবই জনপ্রিয়। এটি ইন্সটল করার পর, তুমি রিয়াক্ট প্রজেক্ট তৈরি করতে প্রস্তুত।
  </Step>

  <Step>
    এরপর, কমান্ড প্রম্পট বা টার্মিনাল খুলো এবং নিচের কমান্ডটি ব্যবহার করো রিয়াক্ট ইন্সটল করতে:

    ```bash
    npm create vite@latest --template react
    ```

    এই কমান্ডটি Vite নামক একটি টুল ব্যবহার করে নতুন রিয়াক্ট প্রজেক্ট তৈরি করবে। Vite হল একটি দ্রুত ডেভেলপমেন্ট পরিবেশ যা রিয়াক্টের জন্য উপযুক্ত।

  </Step>

  <Step>
    কমান্ডটি রান করার পর, তোমাকে তোমার প্রজেক্টের নাম উল্লেখ করতে বলা হবে। তোমার পছন্দ অনুযায়ী একটি নাম দাও এবং এন্টার চাপো। উদাহরণস্বরূপ, তুমি `my-react-app` নাম দিতে পারো।

    এরপর, প্রজেক্ট তৈরি হয়ে গেলে, প্রজেক্টের ডিরেক্টরিতে প্রবেশ করো:

    ```bash
    cd my-react-app
    ```

  </Step>

  <Step>
    অবশেষে, তোমার রিয়াক্ট অ্যাপ্লিকেশন চালু করতে নিচের কমান্ডটি ব্যবহার করো:

    ```bash
    npm install
    npm run dev
    ```

    `npm install` কমান্ডটি প্রয়োজনীয় প্যাকেজগুলো ইনস্টল করবে এবং `npm run dev` কমান্ডটি তোমার অ্যাপ্লিকেশনকে লোকাল সার্ভারে চালু করবে। তোমার ব্রাউজারে গিয়ে `http://localhost:5173` এ ভিজিট করে তোমার নতুন রিয়াক্ট অ্যাপ্লিকেশন দেখতে পারবে।

  </Step>
</Steps>

### Command Follow-Up

তোমার প্রজেক্ট সফলভাবে তৈরি হওয়ার পর, সবগুলো স্টেপ অনুসরণ করো এবং উপরের কমান্ডগুলো চালিয়ে তোমার রিয়াক্ট অ্যাপ্লিকেশন রান করো। যদি সবকিছু ঠিকঠাকভাবে হয়, তাহলে তুমি তোমার ব্রাউজারে রিয়াক্ট অ্যাপ্লিকেশন দেখতে পাবে।
