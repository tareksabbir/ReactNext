# <h1 className="bg-gradient-to-r from-blue-500 to-purple-500 inline-block text-transparent bg-clip-text text-5xl pt-2">React কি?</h1>

রিয়াক্ট হলো ফেসবুকের ডেভেলপ করা একটি জাভাস্ক্রিপ্ট লাইব্রেরি, যা একই সাথে নেটিভ এবং ওয়েব ইন্টারফেস ডিজাইন করতে সহায়তা করে। এটি জাভাস্ক্রিপ্টে লেখা এবং মূলত ইউজার ইন্টারফেস (UI) তৈরি করার জন্য ব্যবহৃত হয়। রিয়াক্টের মাধ্যমে ডেভেলপাররা UI-কে বিভিন্ন কম্পোনেন্টে ভাগ করতে পারেন, যা তাদের কোড পুনঃব্যবহার এবং ব্যবস্থাপনা সহজ করে।

## রিয়াক্টের ইতিহাস

---

ফেসবুকের সফটওয়্যার ইঞ্জিনিয়ার Jordan Walke ২০১১ সালে PHP এর একটি কম্পোনেন্ট-বেসড ফ্রেমওয়ার্ক XHP-js থেকে অনুপ্রাণিত হয়ে React.js তৈরি করেন। এটি প্রথম ফেসবুকের নিউজফিডে ২০১১ সালে ব্যবহার করা হয় এবং ২০১২ সালে Instagram এ ইমপ্লিমেন্ট করা হয়। ২০১৩ সালে React.js কে ওপেন সোর্স হিসেবে প্রকাশ করা হয়। রিয়াক্টের গিটহাব রিপোজিটরি [React.js](https://github.com/facebook/react) এ রয়েছে, যেখানে এটি বিশ্বের বিভিন্ন ডেভেলপারদের দ্বারা অবদান গ্রহণ করছে।

## কেন `React` ব্যবহার করা উচিত?

---

ভ্যানিলা জাভাস্ক্রিপ্টে কাজ করার সময় ইউজার ইন্টারেকশনের ফলে UI পরিবর্তন করতে হলে ডেভেলপারদের ম্যানুয়ালি প্রতিটি পরিবর্তন করতে হয়। এতে করে কোডের জটিলতা এবং উন্নয়নের সময়সীমা বৃদ্ধি পায়।

`React` এর মাধ্যমে ইউজারের ইন্টারেকশনের কারণে যখনই কোন ডেটা বা স্টেট পরিবর্তন হয়, তখন UI টা অটোমেটিক রিয়াক্ট করে। এই অটোমেটিক আপডেটিং প্রক্রিয়াটি UI-কে আরো গতিশীল এবং ব্যবহারকারী-বান্ধব করে তোলে।

## `React` এবং `React DOM` এর ভূমিকা

---

### একটি UI তৈরির জন্য মূলত তিনটি কাজ করতে হয়:

1. **UI তৈরি করা:** কম্পোনেন্টের মাধ্যমে UI ডিজাইন এবং নির্মাণ করা।
2. **ইউজারের ইন্টারেকশন হ্যান্ডেল করা:** ইউজারের ইনপুটের উপর ভিত্তি করে UI পরিবর্তন করা।
3. **রেন্ডারিং:** ব্রাউজারে UI প্রদর্শন করা।

`React` ইউজার ইন্টারফেস তৈরি এবং ডেটা ব্যবস্থাপনায় সাহায্য করে, যেখানে `React DOM` ব্রাউজারে UI রেন্ডার করতে সহায়তা করে। রিয়াক্ট কম্পোনেন্ট ভিত্তিক আর্কিটেকচার ব্যবহার করে, যা ডেভেলপারদের জন্য UI কে ছোট ছোট অংশে বিভক্ত করা সহজ করে।

## কম্পোনেন্ট ভিত্তিক আর্কিটেকচার

---

রিয়াক্টে UI তৈরি করতে কম্পোনেন্ট ব্যবহার করা হয়। প্রতিটি কম্পোনেন্ট একটি নির্দিষ্ট ফাংশন বা দায়িত্ব পালন করে। উদাহরণস্বরূপ:

```javascript
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

// ব্যবহার:
<Welcome name="শেখ মুজিব" />;
```

এই উদাহরণে, `Welcome` কম্পোনেন্ট একটি নাম প্রদর্শন করে, যা প্রপসের মাধ্যমে পাস করা হয়।

### স্টেট এবং প্রপস

React এ ডেটা পরিচালনার জন্য স্টেট এবং প্রপস ব্যবহার করা হয়:

- **স্টেট:** এটি একটি কম্পোনেন্টের অভ্যন্তরীণ ডেটা। যখন স্টেট পরিবর্তন হয়, UI স্বয়ংক্রিয়ভাবে আপডেট হয়। উদাহরণ:

```javascript
function Counter() {
  const [count, setCount] = React.useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  );
}
```

- **প্রপস:** এটি প্যারেন্ট কম্পোনেন্ট থেকে চাইল্ড কম্পোনেন্টে পাস করা ডেটা। প্রপস পরিবর্তন করা যায় না, বরং চাইল্ড কম্পোনেন্ট প্যারেন্টের মাধ্যমে ডেটা গ্রহণ করে।

### উপসংহার

React ব্যবহার করে একটি বেশি ইন্টারেক্টিভ এবং ব্যবহারকারী-বান্ধব UI তৈরি করা যায়। এর কম্পোনেন্ট ভিত্তিক নকশা, অটোমেটিক UI আপডেটের সুবিধা, এবং সহজ ডেটা ব্যবস্থাপনা নিশ্চিত করে। React আধুনিক ওয়েব ডেভেলপমেন্টের জন্য একটি জনপ্রিয় পছন্দ এবং এটি ডেভেলপারদের কোডকে আরও বেশি পরিচালনাযোগ্য এবং কার্যকরী করে তোলে, ফলে এটি আজকের দিনেও একটি অমূল্য টুল হিসেবে বিবেচিত হয়।
