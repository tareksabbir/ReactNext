## <h1 className="bg-gradient-to-r from-blue-500 to-purple-500 inline-block text-transparent bg-clip-text text-3xl pt-2"> Manipulating DOM with Refs</h1>

রিয়াক্ট দিয়ে আমরা সাধারণত DOM (মানে পেজের এলিমেন্ট) নিয়ে কাজ করতে হলে নিজেরা কিছু করতে হয় না, রিয়াক্টই কাজগুলো করে ফেলে। কিন্তু কিছু কিছু সময় এমন হয়, যখন আমাদের নিজের হাতে DOM নিয়ে সরাসরি কিছু কাজ করতে হয়। যেমন ধরো, কোনো এলিমেন্টে সরাসরি ফোকাস দেওয়া, স্ক্রল করে বিশেষ জায়গায় নিয়ে যাওয়া, বা কোনো এলিমেন্টের মাপ মাপা। রিয়াক্টের সাধারণ কাজের ধরণে এসব করার জন্য সরাসরি কোনো অপশন নেই, তাই এইসব ক্ষেত্রে `useRef` নামের একটা হুক (বিশেষ টুল) কাজে আসে।

চল, ধাপে ধাপে দেখি কীভাবে `useRef` দিয়ে DOM মডিফাই করতে পারি।

---

### DOM মডিফাই করার স্টেপগুলো

#### Step 1: রিয়াক্ট থেকে `useRef` ইম্পোর্ট করা

প্রথমে, আমরা `useRef` ইম্পোর্ট করে নিবো। কারণ, এটা রিয়াক্টে বিল্ট-ইন, বাইরের কোনো কিছু না।

```jsx
import { useRef } from "react";
```

#### Step 2: কম্পোনেন্টের ভেতরে একটা রেফ তৈরি করা

এখন, আমরা কম্পোনেন্টের ভেতরে একটা রেফ তৈরি করবো। এটা এমন একটা জিনিস যেটার মাধ্যমে আমরা আমাদের DOM এলিমেন্টটাকে ধরতে পারবো। প্রথমে আমরা রেফের নাম দিলাম `myRef` এবং এটাকে `null` দিয়ে শুরু করলাম। এর মানে হলো, আমরা এখনো এই রেফ দিয়ে কিছু ধরিনি।

```jsx
const myRef = useRef(null);
```

#### Step 3: রেফটাকে যেই এলিমেন্টে অ্যাপ্লাই করবো সেটাতে `ref` হিসেবে সেট করা

এখন আমরা সেই এলিমেন্টটাকে `ref` হিসেবে ধরবো যেটাতে DOM মডিফাই করতে চাই। এজন্য আমাদের HTML ট্যাগের মতো যেকোনো এলিমেন্টে `ref` দিয়ে এটাকে সেট করবো।

```jsx
<div ref={myRef}>এটা এমনিতে দেখাচ্ছে</div>
<button onClick={handleChangeText}>টেক্সট বদলাও</button>
```

#### Step 4: ইভেন্ট হ্যান্ডলারে গিয়ে DOM মডিফাই করা

এখন `myRef` এর সাহায্যে আমরা `div` এলিমেন্টটাতে সরাসরি কাজ করতে পারবো। ধরো আমরা এলিমেন্টটাতে টেক্সট বদলাতে চাই; তাহলে আমরা একটা `handleChangeText` নামে ফাংশন বানিয়ে সেখানে `myRef.current` দিয়ে এলিমেন্টটাতে নতুন টেক্সট সেট করতে পারি।

```jsx
function handleChangeText() {
    myRef.current.innerText = "এই টেক্সটটা এখন রেফের মাধ্যমে আসছে!";
}
```

### পুরো কোডটা

এবার পুরো কোডটা একসাথে দেখে নিই যেন পুরোটা বোঝা যায়। 

```jsx
import { useRef } from "react";

export default function Example() {
  const myRef = useRef(null);

  function handleChangeText() {
    myRef.current.innerText = "এই টেক্সটটা এখন রেফের মাধ্যমে আসছে!";
  }

  return (
    <>
      <div ref={myRef}>এটা এমনিতে দেখাচ্ছে</div>
      <button onClick={handleChangeText}>টেক্সট বদলাও</button>
    </>
  );
}
```

এখানে যখন আমরা বাটনে ক্লিক করবো, তখন `div` এর টেক্সট বদলে যাবে কারণ আমরা `myRef` দিয়ে সরাসরি সেই DOM এলিমেন্টটাতে ঢুকে গেছি।
