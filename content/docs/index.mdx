# React & Next.js ডকুমেন্টেশন
-----

React Next এ আপনাকে স্বাগতম! আপনি যদি একজন React ডেভেলপার হতে চান, তাহলে এই ডকুমেন্টেশনটি আপনার জন্য অত্যন্ত উপকারী হবে। এখানে আপনি ধাপে ধাপে React এবং Next.js খুব সহজেই বাংলা ভাষায় শিখতে পারবেন।

React এ যেতে হলে আপনাকে প্রথমে কিছু বেসিক বিষয় সম্পর্কে জানতে হবে যেমন Git, GitHub, এবং Basic JavaScript। আপনি দয়া করে তাদের অফিসিয়াল ডকুমেন্টেশনগুলো দেখে নিন, সেগুলো ভালোভাবে শিখে নিন। তারপর আমরা ধাপে ধাপে React শিখব।

এই ডকুমেন্টেশনটি আপনাকে সাহায্য করবে React এর জগতে দক্ষ হওয়ার জন্য।
